**Project Type:** Educational Tutorial / Self-Learning Materials

## Key HTML Concepts Taught

| Concept | Tags Used | Purpose |
|---------|-----------|---------|
| Structure | `<!DOCTYPE>`, `<html>`, `<head>`, `<body>` | Define document framework |
| Headings | `<h1>` to `<h6>` | Create section titles |
| Content | `<p>`, `<br>` | Structure text and paragraphs |
| Lists | `<ul>`, `<ol>`, `<li>` | Create bullet and numbered lists |
| Links | `<a>` | Create hyperlinks |
| Media | `<img>` | Embed images |
| Formatting | `<strong>`, `<em>`, `<u>` | Style text |
| Layout | `<hr>` | Add visual separators |

---

## Important Rules Highlighted in Tutorial

1. **Tags must be paired** - Every opening tag needs a closing tag
2. **Some tags are self-closing** - `<br>`, `<hr>`, `<img>` don't need closing tags
3. **Case insensitive** - HTML tags work in uppercase or lowercase (lowercase is preferred)
4. **Indentation matters** - Proper formatting makes code easier to read and maintain
